# Bolo Sem Glúten - Clone Perfeito

Este é o clone completo e perfeito do site [Receitas de Bolo Sem Glúten](https://bolo-semgluten.netlify.app/).

## Estrutura do Projeto

O projeto foi estruturado da seguinte forma:
- `index.html` - O arquivo principal contendo a estrutura da página, CSS crítico inline e scripts de animação/rastreamento.
- `css/style.css` - Estilos não-críticos contendo designs modernos e variações de componentes do site.
- `fonts/` - Fontes otimizadas usadas no design do site (`Asap Condensed`, `Raleway`).
- `images/` - Todas as imagens e assets gráficos necessários para o site carregar e rodar perfeitamente (mockups, fotos de bolos, ícones de ingredientes, garantia, etc.).

## Como Visualizar Localmente

Como todos os caminhos de arquivos e links internos foram convertidos para caminhos relativos (em vez de caminhos absolutos baseados em servidor), você pode abrir o site de várias maneiras:

1. **Diretamente no navegador**:
   Basta dar um duplo clique no arquivo [index.html](file:///C:/Users/Usuario/.gemini/antigravity/scratch/bolo-semgluten-clone/index.html) e ele se abrirá diretamente no seu navegador utilizando o protocolo local (`file://`). Todas as imagens e fontes serão carregadas corretamente!

2. **Usando um Servidor Local Simples**:
   Se você tiver o Python instalado, abra o terminal no diretório do projeto e execute:
   ```bash
   python -m http.server 8000
   ```
   Depois acesse `http://localhost:8000` no seu navegador.
